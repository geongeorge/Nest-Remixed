# Nest Remixed

NestJS starter with Remix frontend. This repo is a yarn workspace with two packages: `server` and `remix`

## Features

- Remix js HMR
- Watch mode for NestJS
- One server for both frontend and backend
- Yarn workspace

## Installation

```bash
yarn
```

## Running the app

```bash
yarn dev
```
